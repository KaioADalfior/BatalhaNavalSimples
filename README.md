
Este é um projetinho que desenvolvi para colocar em prática o que estou aprendendo em Java. 
Decidi recriar o clássico jogo de Batalha Naval para rodar direto no terminal, focando puramente na lógica e na organização do código.
O sistema gera um tabuleiro de 5x5 e esconde 3 navios em posições aleatórias.
